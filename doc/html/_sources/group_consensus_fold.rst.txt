        float energy_of_alistruct (
	    const char** sequences,
	    const char* structure,
	    int n_seq,
	    float* energy
	    )

    .. container:: brief-description

        Calculate the free energy of a consensus structure given a set of aligned sequences.

    .. container:: detailed-description

        Deprecated Usage of this function is discouraged! Use :ref:`vrna_eval_structure() <doxid-group__eval_1ga58f199f1438d794a265f3b27fc8ea631>` , and :ref:`vrna_eval_covar_structure() <doxid-group__eval_1ga6cea75c0eb9857fb59172be54cab09e0>` instead!
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - sequences
        
                - The NULL terminated array of sequences
        
            *
                - structure
        
                - The consensus structure
        
            *
                - n_seq
        
                - The number of sequences in the alignment
        
            *
                - energy
        
                - A pointer to an array of at least two floats that will hold the free energies (energy[0] will contain the free energy, energy[1] will be filled with the covariance energy term)
        
        
        
        .. rubric:: Returns:
        
        free energy in kcal/mol

        int get_alipf_arrays (
	    short*** S_p,
	    short*** S5_p,
	    short*** S3_p,
	    unsigned short*** a2s_p,
	    char*** Ss_p,
	    :ref:`FLT_OR_DBL<doxid-group__data__structures_1ga31125aeace516926bf7f251f759b6126>`** qb_p,
	    :ref:`FLT_OR_DBL<doxid-group__data__structures_1ga31125aeace516926bf7f251f759b6126>`** qm_p,
	    :ref:`FLT_OR_DBL<doxid-group__data__structures_1ga31125aeace516926bf7f251f759b6126>`** q1k_p,
	    :ref:`FLT_OR_DBL<doxid-group__data__structures_1ga31125aeace516926bf7f251f759b6126>`** qln_p,
	    short** pscore
	    )

    .. container:: brief-description

        Get pointers to (almost) all relavant arrays used in alifold's partition function computation.

    .. container:: detailed-description

        Deprecated It is discouraged to use this function! The new :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` allows direct access to all necessary consensus structure prediction related variables!
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - S_p
        
                - A pointer to the 'S' array (integer representation of nucleotides)
        
            *
                - S5_p
        
                - A pointer to the 'S5' array
        
            *
                - S3_p
        
                - A pointer to the 'S3' array
        
            *
                - a2s_p
        
                - A pointer to the alignment-column to sequence position mapping array
        
            *
                - Ss_p
        
                - A pointer to the 'Ss' array
        
            *
                - qb_p
        
                - A pointer to the Q B matrix
        
            *
                - qm_p
        
                - A pointer to the Q M matrix
        
            *
                - q1k_p
        
                - A pointer to the 5' slice of the Q matrix ( :math:`q1k(k) = Q(1, k)` )
        
            *
                - qln_p
        
                - A pointer to the 3' slice of the Q matrix ( :math:`qln(l) = Q(l, n)` )
        
            *
                - pscore
        
                - A pointer to the start of a pscore list
        
        
        
        .. rubric:: Returns:
        
        Non Zero if everything went fine, 0 otherwise
        
        
        
        .. note::
        
            To obtain meaningful pointers, call alipf_fold first!
        
        
        
        .. admonition:: See also:
        
            pf_alifold(), :ref:`alipf_circ_fold() <doxid-group__consensus__pf__fold_1ga604a42ad64178279551ad3e4def3d603>`
        
            :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` , :ref:`vrna_fold_compound_comparative() <doxid-group__fold__compound_1gad6bacc816af274922b13d947f708aa0c>` , :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>`
