        int rna_plot_type

    .. container:: brief-description

        Switch for changing the secondary structure layout algorithm.

    .. container:: detailed-description

        Current possibility are 0 for a simple radial drawing or 1 for the modified radial drawing taken from the *naview* program of :ref:`[3] <doxid-citelist_1CITEREF_bruccoleri:1988>` .
        
        
        
        .. note::
        
            To provide thread safety please do not rely on this global variable in future implementations but pass a plot type flag directly to the function that decides which layout algorithm it may use!
        
        
        
        .. admonition:: See also:
        
            :ref:`VRNA_PLOT_TYPE_SIMPLE <doxid-group__plotting__utils_1gae6d17b9f0a53cf5205a9181e0f8422e9>` , :ref:`VRNA_PLOT_TYPE_NAVIEW <doxid-group__plotting__utils_1ga94d4c863ecac2f220f76658afb92f964>` , :ref:`VRNA_PLOT_TYPE_CIRCULAR <doxid-group__plotting__utils_1ga8c9eac631348da92136c8363ecdd9fb9>`

    .. ref-code-block:: cpp
        :class: title-code-block

        int simple_circplot_coordinates (
	    short* pair_table,
	    float* x,
	    float* y
	    )

    .. container:: brief-description

        Calculate nucleotide coordinates for *Circular Plot*

    .. container:: detailed-description

        This function calculates the coordinates of nucleotides mapped in equal distancies onto a unit circle.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - pair_table
        
                - The pair table of the secondary structure
        
            *
                - x
        
                - a pointer to an array with enough allocated space to hold the x coordinates
        
            *
                - y
        
                - a pointer to an array with enough allocated space to hold the y coordinates
        
        
        
        .. rubric:: Returns:
        
        length of sequence on success, 0 otherwise
        
        
        
        .. note::
        
            In order to draw nice arcs using quadratic bezier curves that connect base pairs one may calculate a second tangential point :math:`P^t` in addition to the actual R 2 coordinates. the simplest way to do so may be to compute a radius scaling factor :math:`rs` in the interval :math:`[0,1]` that weights the proportion of base pair span to the actual length of the sequence. This scaling factor can then be used to calculate the coordinates for :math:`P^t` , i.e. :math:` P^{t}_x[i] = X[i] * rs` and :math:`P^{t}_y[i] = Y[i] * rs` .
        
        
        
        .. admonition:: See also:
        
            :ref:`make_pair_table() <doxid-group__struct__utils_1ga89c32307ee50a0026f4a3131fac0845a>` , :ref:`rna_plot_type <doxid-group__plotting__utils_1ga5964c4581431b098b80027d6e14dcdd4>` , :ref:`simple_xy_coordinates() <doxid-group__plotting__utils_1gaf4b9173e7d3fd361c3c85e6def194123>` , :ref:`naview_xy_coordinates() <doxid-group__plotting__utils_1ga948111b910926f0405a3fa920c790816>` , :ref:`vrna_file_PS_rnaplot_a() <doxid-plot__structure_8h_1a139a31dd0ba9fc6612431f67de901c31>` , :ref:`vrna_file_PS_rnaplot <doxid-plot__structure_8h_1abdc8f6548ba4a3bc3cd868ccbcfdb86a>` , :ref:`svg_rna_plot() <doxid-plot__structure_8h_1ae7853539b5df98f294b4af434e979304>`

    .. ref-code-block:: cpp
        :class: title-code-block

        int PS_dot_plot_list (
	    char* seq,
	    char* filename,
	    :ref:`plist<doxid-group__data__structures_1ga9608eed021ebfbdd7a901cfdc446c8e9>`* pl,
	    :ref:`plist<doxid-group__data__structures_1ga9608eed021ebfbdd7a901cfdc446c8e9>`* mf,
	    char* comment
	    )

    .. container:: brief-description

        Produce a postscript dot-plot from two pair lists.

    .. container:: detailed-description

        This function reads two plist structures (e.g. base pair probabilities and a secondary structure) as produced by :ref:`assign_plist_from_pr() <doxid-group__pf__fold_1ga1cc05aaa9b0e7df2d3887e98321c2030>` and :ref:`assign_plist_from_db() <doxid-group__struct__utils_1gab61df77cf7949cd516181fce0c3d7d78>` and produces a postscript "dot plot" that is written to 'filename'.
        
        Using base pair probabilities in the first and mfe structure in the second plist, the resulting "dot plot" represents each base pairing probability by a square of corresponding area in a upper triangle matrix. The lower part of the matrix contains the minimum free energy structure.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - seq
        
                - The RNA sequence
        
            *
                - filename
        
                - A filename for the postscript output
        
            *
                - pl
        
                - The base pair probability pairlist
        
            *
                - mf
        
                - The mfe secondary structure pairlist
        
            *
                - comment
        
                - A comment
        
        
        
        .. rubric:: Returns:
        
        1 if postscript was successfully written, 0 otherwise
        
        
        
        .. admonition:: See also:
        
            :ref:`assign_plist_from_pr() <doxid-group__pf__fold_1ga1cc05aaa9b0e7df2d3887e98321c2030>` , :ref:`assign_plist_from_db() <doxid-group__struct__utils_1gab61df77cf7949cd516181fce0c3d7d78>`
