    .. container:: brief-description

        Computes a MEA (maximum expected accuracy) structure.

    .. container:: detailed-description

        The algorithm maximizes the expected accuracy
        
        .. math::
        
            A(S) = \sum_{(i,j) \in S} 2 \gamma p_{ij} + \sum_{i \notin S} p^u_i
        
        Higher values of :math:`\gamma` result in more base pairs of lower probability and thus higher sensitivity. Low values of :math:`\gamma` result in structures containing only highly likely pairs (high specificity). The code of the MEA function also demonstrates the use of sparse dynamic programming scheme to reduce the time and memory complexity of folding.
