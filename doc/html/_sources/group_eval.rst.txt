        float vrna_eval_structure (
	    :ref:`vrna_fold_compound_t<doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>`* vc,
	    const char* structure
	    )

    .. container:: brief-description

        Calculate the free energy of an already folded RNA.

    .. container:: detailed-description

        This function allows for energy evaluation of a given pair of structure and sequence (alignment). Model details, energy parameters, and possibly soft constraints are used as provided via the parameter 'vc'. The :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` does not need to contain any DP matrices, but requires all most basic init values as one would get from a call like this:
        
        .. ref-code-block:: cpp
        
            vc = :ref:`vrna_fold_compound <doxid-group__fold__compound_1ga6601d994ba32b11511b36f68b08403be>`(sequence, NULL, :ref:`VRNA_OPTION_EVAL_ONLY <doxid-group__fold__compound_1ga61469c423131552c8483229f8b6c7e0e>`);
        
        SWIG Wrapper Notes This function is attached as method **eval_structure()** to objects of type *fold_compound*
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - vc
        
                - A vrna_fold_compound_t containing the energy parameters and model details
        
            *
                - structure
        
                - Secondary structure in dot-bracket notation
        
        
        
        .. rubric:: Returns:
        
        The free energy of the input structure given the input sequence in kcal/mol
        
        
        
        .. note::
        
            Accepts vrna_fold_compound_t of type :ref:`VRNA_FC_TYPE_SINGLE <doxid-group__fold__compound_1gga01a4ff86fa71deaaa5d1abbd95a1447da7e264dd3cf2dc9b6448caabcb7763cd6>` and :ref:`VRNA_FC_TYPE_COMPARATIVE <doxid-group__fold__compound_1gga01a4ff86fa71deaaa5d1abbd95a1447dab821ce46ea3cf665be97df22a76f5023>`
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_eval_structure_pt() <doxid-group__eval_1gadbd09372ddfd7a450bbd590c96a6bfe4>` , :ref:`vrna_eval_structure_verbose() <doxid-group__eval_1ga0928d699d310178f84ee2351034e5cb5>` , :ref:`vrna_eval_structure_pt_verbose() <doxid-group__eval_1ga8a517cfeeae8c376ae7b1e0c401d38b4>` , :ref:`vrna_fold_compound() <doxid-group__fold__compound_1ga6601d994ba32b11511b36f68b08403be>` , :ref:`vrna_fold_compound_comparative() <doxid-group__fold__compound_1gad6bacc816af274922b13d947f708aa0c>` , :ref:`vrna_eval_covar_structure() <doxid-group__eval_1ga6cea75c0eb9857fb59172be54cab09e0>`

    .. ref-code-block:: cpp

        float vrna_eval_covar_structure (
	    :ref:`vrna_fold_compound_t<doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>`* vc,
	    const char* structure
	    )

    .. container:: brief-description

        Calculate the pseudo energy derived by the covariance scores of a set of aligned sequences.

    .. container:: detailed-description

        Consensus structure prediction is driven by covariance scores of base pairs in rows of the provided alignment. This function allows one to retrieve the total amount of this covariance pseudo energy scores. The :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` does not need to contain any DP matrices, but requires all most basic init values as one would get from a call like this:
        
        .. ref-code-block:: cpp
        
            vc = :ref:`vrna_fold_compound_comparative <doxid-group__fold__compound_1gad6bacc816af274922b13d947f708aa0c>`(alignment, NULL, :ref:`VRNA_OPTION_EVAL_ONLY <doxid-group__fold__compound_1ga61469c423131552c8483229f8b6c7e0e>`);
        
        SWIG Wrapper Notes This function is attached as method **eval_covar_structure()** to objects of type *fold_compound*
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - vc
        
                - A vrna_fold_compound_t containing the energy parameters and model details
        
            *
                - structure
        
                - Secondary (consensus) structure in dot-bracket notation
        
        
        
        .. rubric:: Returns:
        
        The covariance pseudo energy score of the input structure given the input sequence alignment in kcal/mol
        
        
        
        .. note::
        
            Accepts vrna_fold_compound_t of type :ref:`VRNA_FC_TYPE_COMPARATIVE <doxid-group__fold__compound_1gga01a4ff86fa71deaaa5d1abbd95a1447dab821ce46ea3cf665be97df22a76f5023>` only!
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_fold_compound_comparative() <doxid-group__fold__compound_1gad6bacc816af274922b13d947f708aa0c>` , :ref:`vrna_eval_structure() <doxid-group__eval_1ga58f199f1438d794a265f3b27fc8ea631>`

    .. container:: brief-description

        Evaluate free energy of a sequence/structure pair, allow for G-Quadruplexes in the structure and print contributions per loop.

    .. container:: detailed-description

        This function is the same as :ref:`vrna_eval_structure_simple_v() <doxid-group__eval_1gacd6278343e77d13f1d53588e50d303bc>` but allows for annotated G-Quadruplexes in the dot-bracket structure input.
        
        G-Quadruplexes are annotated as plus signs ('+') for each G involved in the motif. Linker sequences must be denoted by dots ('.') as they are considered unpaired. Below is an example of a 2-layer G-quadruplex:
        
        .. ref-code-block:: cpp
        
            GGAAGGAAAGGAGG
            ++..++...++.++
        
        SWIG Wrapper Notes This function is available through an overloaded version of :ref:`vrna_eval_gquad_structure() <doxid-group__eval_1ga3263504825ef4b523eba797c99921df4>` . The last two arguments for this function are optional and default to :ref:`VRNA_VERBOSITY_QUIET <doxid-eval_8h_1af4afe19780b61b4962c613bde324128b>` and NULL, respectively.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - string
        
                - RNA sequence in uppercase letters
        
            *
                - structure
        
                - Secondary structure in dot-bracket notation
        
            *
                - verbosity_level
        
                - The level of verbosity of this function
        
            *
                - file
        
                - A file handle where this function should print to (may be NULL).
        
        
        
        .. rubric:: Returns:
        
        The free energy of the input structure given the input sequence in kcal/mol

        float vrna_eval_consensus_structure_simple (
	    const char** alignment,
	    const char* structure
	    )

    .. container:: brief-description

        Calculate the free energy of an already folded RNA sequence alignment.

    .. container:: detailed-description

        This function allows for energy evaluation for a given multiple sequence alignment and consensus structure pair. In contrast to :ref:`vrna_eval_structure() <doxid-group__eval_1ga58f199f1438d794a265f3b27fc8ea631>` this function assumes default model details and default energy parameters in order to evaluate the free energy of the secondary structure. Therefore, it serves as a simple interface function for energy evaluation for situations where no changes on the energy model are required.
        
        SWIG Wrapper Notes This function is available through an overloadeded version of :ref:`vrna_eval_structure_simple() <doxid-group__eval_1ga7e5273464b775d4130245681312c1369>` . Simply pass a sequence alignment as list of strings (including gaps) as first, and the consensus structure as second argument
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - alignment
        
                - RNA sequence alignment in uppercase letters and hyphen ('-') to denote gaps
        
            *
                - structure
        
                - Consensus Secondary structure in dot-bracket notation
        
        
        
        .. rubric:: Returns:
        
        The free energy of the consensus structure given the input alignment in kcal/mol
        
        
        
        .. note::
        
            The free energy returned from this function already includes the covariation pseudo energies that is used fir comparative structure prediction within this library.
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_eval_covar_structure() <doxid-group__eval_1ga6cea75c0eb9857fb59172be54cab09e0>` , :ref:`vrna_eval_structure() <doxid-group__eval_1ga58f199f1438d794a265f3b27fc8ea631>` , :ref:`vrna_eval_structure_pt() <doxid-group__eval_1gadbd09372ddfd7a450bbd590c96a6bfe4>` , :ref:`vrna_eval_structure_verbose() <doxid-group__eval_1ga0928d699d310178f84ee2351034e5cb5>` , :ref:`vrna_eval_structure_pt_verbose() <doxid-group__eval_1ga8a517cfeeae8c376ae7b1e0c401d38b4>`
