Detailed Documentation
----------------------

.. container:: brief-description

    The hard constraints data structure.

.. container:: detailed-description

    The content of this data structure determines the decomposition pattern used in the folding recursions. Attribute 'matrix' is used as source for the branching pattern of the decompositions during all folding recursions. Any entry in matrix[i,j] consists of the 6 LSB that allows one to distinguish the following types of base pairs:
    
    * in the exterior loop ( :ref:`VRNA_CONSTRAINT_CONTEXT_EXT_LOOP <doxid-group__hard__constraints_1ga9418eda62a5dec070896702c279d2548>` )
    
    * enclosing a hairpin ( :ref:`VRNA_CONSTRAINT_CONTEXT_HP_LOOP <doxid-group__hard__constraints_1ga79203702b197b6b9d3b78eed40663eb1>` )
    
    * enclosing an interior loop ( :ref:`VRNA_CONSTRAINT_CONTEXT_INT_LOOP <doxid-group__hard__constraints_1ga21feeab3a9e5fa5a9e3d9ac0fcf5994f>` )
    
    * enclosed by an exterior loop ( :ref:`VRNA_CONSTRAINT_CONTEXT_INT_LOOP_ENC <doxid-group__hard__constraints_1ga0536288e04ff6332ecdc23ca4705402b>` )
    
    * enclosing a multi branch loop ( :ref:`VRNA_CONSTRAINT_CONTEXT_MB_LOOP <doxid-group__hard__constraints_1ga456ecd2ff00056bb64da8dd4f61bbfc5>` )
    
    * enclosed by a multi branch loop ( :ref:`VRNA_CONSTRAINT_CONTEXT_MB_LOOP_ENC <doxid-group__hard__constraints_1ga02a3d703ddbcfce393e4bbfcb9db7077>` )
    
    The four linear arrays 'up_xxx' provide the number of available unpaired nucleotides (including position i) 3' of each position in the sequence.
    
    
    
    .. admonition:: See also:
    
        :ref:`vrna_hc_init() <doxid-group__hard__constraints_1ga36ff456c43bf920629cee5a236e4f0ff>` , :ref:`vrna_hc_free() <doxid-group__hard__constraints_1ga696dcf77887d856c6f21ea266d8b9ca2>` , :ref:`VRNA_CONSTRAINT_CONTEXT_EXT_LOOP <doxid-group__hard__constraints_1ga9418eda62a5dec070896702c279d2548>` , :ref:`VRNA_CONSTRAINT_CONTEXT_HP_LOOP <doxid-group__hard__constraints_1ga79203702b197b6b9d3b78eed40663eb1>` , :ref:`VRNA_CONSTRAINT_CONTEXT_INT_LOOP <doxid-group__hard__constraints_1ga21feeab3a9e5fa5a9e3d9ac0fcf5994f>` , :ref:`VRNA_CONSTRAINT_CONTEXT_MB_LOOP <doxid-group__hard__constraints_1ga456ecd2ff00056bb64da8dd4f61bbfc5>` , :ref:`VRNA_CONSTRAINT_CONTEXT_MB_LOOP_ENC <doxid-group__hard__constraints_1ga02a3d703ddbcfce393e4bbfcb9db7077>`
