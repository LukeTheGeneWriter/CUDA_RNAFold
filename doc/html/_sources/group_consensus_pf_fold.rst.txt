        float vrna_pf_alifold (
	    const char** sequences,
	    char* structure,
	    :ref:`vrna_ep_t<doxid-group__struct__utils_1gab9ac98ab55ded9fb90043b024b915aca>`** pl
	    )

    .. container:: brief-description

        Compute Partition function :math:`Q` (and base pair probabilities) for an RNA sequence alignment using a comparative method.

    .. container:: detailed-description

        This simplified interface to :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` computes the partition function and, if required, base pair probabilities for an RNA sequence alignment using default options. Memory required for dynamic programming (DP) matrices will be allocated and free'd on-the-fly. Hence, after return of this function, the recursively filled matrices are not available any more for any post-processing.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - sequences
        
                - RNA sequence alignment
        
            *
                - structure
        
                - A pointer to the character array where position-wise pairing propensity will be stored. (Maybe NULL)
        
            *
                - pl
        
                - A pointer to a list of :ref:`vrna_ep_t <doxid-group__struct__utils_1gab9ac98ab55ded9fb90043b024b915aca>` to store pairing probabilities (Maybe NULL)
        
        
        
        .. rubric:: Returns:
        
        The Gibbs free energy of the ensemble ( :math:`G = -RT \cdot \log(Q) ` ) in kcal/mol
        
        
        
        .. note::
        
            In case you want to use the filled DP matrices for any subsequent post-processing step, or you require other conditions than specified by the default model details, use :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` , and the data structure :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` instead.
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_pf_circalifold() <doxid-group__consensus__pf__fold_1gab70fe6c9a78b79cc5669881720926e1d>` , :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` , :ref:`vrna_fold_compound_comparative() <doxid-group__fold__compound_1gad6bacc816af274922b13d947f708aa0c>` , :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>`

        float vrna_pf_circalifold (
	    const char** sequences,
	    char* structure,
	    :ref:`vrna_ep_t<doxid-group__struct__utils_1gab9ac98ab55ded9fb90043b024b915aca>`** pl
	    )

    .. container:: brief-description

        Compute Partition function :math:`Q` (and base pair probabilities) for an alignment of circular RNA sequences using a comparative method.

    .. container:: detailed-description

        This simplified interface to :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` computes the partition function and, if required, base pair probabilities for an RNA sequence alignment using default options. Memory required for dynamic programming (DP) matrices will be allocated and free'd on-the-fly. Hence, after return of this function, the recursively filled matrices are not available any more for any post-processing.
        
        Folding of circular RNA sequences is handled as a post-processing step of the forward recursions. See :ref:`[7] <doxid-citelist_1CITEREF_hofacker:2006>` for further details.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - sequences
        
                - Sequence alignment of circular RNAs
        
            *
                - structure
        
                - A pointer to the character array where position-wise pairing propensity will be stored. (Maybe NULL)
        
            *
                - pl
        
                - A pointer to a list of :ref:`vrna_ep_t <doxid-group__struct__utils_1gab9ac98ab55ded9fb90043b024b915aca>` to store pairing probabilities (Maybe NULL)
        
        
        
        .. rubric:: Returns:
        
        The Gibbs free energy of the ensemble ( :math:`G = -RT \cdot \log(Q) ` ) in kcal/mol
        
        
        
        .. note::
        
            In case you want to use the filled DP matrices for any subsequent post-processing step, or you require other conditions than specified by the default model details, use :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` , and the data structure :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` instead.
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_pf_alifold() <doxid-group__consensus__pf__fold_1ga374e31a0f326b2c5da5b84e143a63f38>` , :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` , :ref:`vrna_fold_compound_comparative() <doxid-group__fold__compound_1gad6bacc816af274922b13d947f708aa0c>` , :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>`

The list is terminated by the first entry with pi.i = 0.

        :ref:`FLT_OR_DBL<doxid-group__data__structures_1ga31125aeace516926bf7f251f759b6126>`* export_ali_bppm (void)

    .. container:: brief-description

        Get a pointer to the base pair probability array.

    .. container:: detailed-description

        Accessing the base pair probabilities for a pair (i,j) is achieved by
        
        .. code-block:: none
        
            FLT_OR_DBL *pr = export_bppm(); pr_ij = pr[iindx[i]-j];
        
        Deprecated Usage of this function is discouraged! The new :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` allows direct access to the folding matrices, including the pair probabilities! The pair probability array returned here reflects the one of the latest call to :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` , or any of the old API calls for consensus structure partition function folding.
        
        
        
        .. rubric:: Returns:
        
        A pointer to the base pair probability array
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` , :ref:`vrna_fold_compound_comparative() <doxid-group__fold__compound_1gad6bacc816af274922b13d947f708aa0c>` , and :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>`
