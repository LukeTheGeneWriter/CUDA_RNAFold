.. container:: typedef-container

    .. ref-code-block:: cpp
        :class: title-code-block

        typedef void () vrna_callback_stream_output (
	    void *auxdata,
	    unsigned int i,
	    void *data
	    )

    .. container:: brief-description

        Ordered stream processing callback.

    .. container:: detailed-description

        This callback will be processed in sequential order as soon as sequential data in the output stream becomes available.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - auxdata
        
                - A shared pointer for all calls, as provided by the second argument to :ref:`vrna_ostream_init() <doxid-group__buffer__utils_1gad23113e66a0910ec2341856e2da56bf6>`
        
            *
                - i
        
                - The index number of the data passed to ``data``
        
            *
                - data
        
                - A block of data ready for processing
        
        
        
        .. note::
        
            The callback must also release the memory occupied by the data passed since the stream will lose any reference to it after the callback has been executed.
