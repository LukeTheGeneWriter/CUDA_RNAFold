    .. ref-code-block:: cpp
        :class: title-code-block

        float* vrna_aln_conservation_struct (
	    const char** alignment,
	    const char* structure,
	    const :ref:`vrna_md_t<doxid-group__model__details_1ga1f8a10e12a0a1915f2a4eff0b28ea17c>`* md
	    )

    .. container:: brief-description

        Compute base pair conservation of a consensus structure.

    .. container:: detailed-description

        This function computes the base pair conservation (fraction of canonical base pairs) of a consensus structure given a multiple sequence alignment. The base pair types that are considered canonical may be specified using the #vrna_md_t.pairs array. Passing *NULL* as parameter ``md`` results in default pairing rules, i.e. canonical Watson-Crick and GU Wobble pairs.
        
        SWIG Wrapper Notes This function is available in an overloaded form where the last parameter may be omitted, indicating ``md`` = *NULL*
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - alignment
        
                - The input sequence alignment (last entry must be *NULL* terminated)
        
            *
                - structure
        
                - The consensus structure in dot-bracket notation
        
            *
                - md
        
                - Model details that specify compatible base pairs (Maybe *NULL* )
        
        
        
        .. rubric:: Returns:
        
        A 1-based vector of base pair conservations

    .. ref-code-block:: cpp
        :class: title-code-block

        float* vrna_aln_conservation_col (
	    const char** alignment,
	    const :ref:`vrna_md_t<doxid-group__model__details_1ga1f8a10e12a0a1915f2a4eff0b28ea17c>`* md_p,
	    unsigned int options
	    )

    .. container:: brief-description

        Compute nucleotide conservation in an alignment.

    .. container:: detailed-description

        This function computes the conservation of nucleotides in alignment columns. The simples measure is Shannon Entropy and can be selected by passing the :ref:`VRNA_MEASURE_SHANNON_ENTROPY <doxid-group__aln__utils_1ga1e659227c9fc077d29989f576f129000>` flag in the ``options`` parameter.
        
        SWIG Wrapper Notes This function is available in an overloaded form where the last two parameters may be omitted, indicating ``md`` = *NULL* , and ``options`` = :ref:`VRNA_MEASURE_SHANNON_ENTROPY <doxid-group__aln__utils_1ga1e659227c9fc077d29989f576f129000>` , respectively.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - alignment
        
                - The input sequence alignment (last entry must be *NULL* terminated)
        
            *
                - md
        
                - Model details that specify known nucleotides (Maybe *NULL* )
        
            *
                - options
        
                - A flag indicating which measure of conservation should be applied
        
        
        
        .. rubric:: Returns:
        
        A 1-based vector of column conservations
        
        
        
        .. note::
        
            Currently, only :ref:`VRNA_MEASURE_SHANNON_ENTROPY <doxid-group__aln__utils_1ga1e659227c9fc077d29989f576f129000>` is supported as conservation measure.
        
        
        
        .. admonition:: See also:
        
            :ref:`VRNA_MEASURE_SHANNON_ENTROPY <doxid-group__aln__utils_1ga1e659227c9fc077d29989f576f129000>`
