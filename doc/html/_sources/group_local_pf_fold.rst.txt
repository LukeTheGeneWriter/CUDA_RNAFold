        :ref:`vrna_ep_t<doxid-group__struct__utils_1gab9ac98ab55ded9fb90043b024b915aca>`* vrna_pfl_fold (
	    const char* sequence,
	    int window_size,
	    int max_bp_span,
	    float cutoff
	    )

    .. container:: brief-description

        Compute base pair probabilities using a sliding-window approach.

    .. container:: detailed-description

        This is a simplified wrapper to :ref:`vrna_probs_window() <doxid-group__local__pf__fold_1ga70ec33810fa17a9f67a7c59f156d0137>` that given a nucleid acid sequence, a window size, a maximum base pair span, and a cutoff value computes the pair probabilities for any base pair in any window. The pair probabilities are returned as a list and the user has to take care to free() the memory occupied by the list.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - sequence
        
                - The nucleic acid input sequence
        
            *
                - window_size
        
                - The size of the sliding window
        
            *
                - max_bp_span
        
                - The maximum distance along the backbone between two nucleotides that form a base pairs
        
            *
                - cutoff
        
                - A cutoff value that omits all pairs with lower probability
        
        
        
        .. rubric:: Returns:
        
        A list of base pair probabilities, terminated by an entry with :ref:`vrna_ep_t.i <doxid-structvrna__elem__prob__s_1a0f8bb11ded4e605f816d7ad92eb568f6>` and :ref:`vrna_ep_t.j <doxid-structvrna__elem__prob__s_1acada5be62ed6843334a918ca543f0c0d>` set to 0
        
        
        
        .. note::
        
            This function uses default model settings! For custom model settings, we refer to the function :ref:`vrna_probs_window() <doxid-group__local__pf__fold_1ga70ec33810fa17a9f67a7c59f156d0137>` .
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_probs_window() <doxid-group__local__pf__fold_1ga70ec33810fa17a9f67a7c59f156d0137>` , :ref:`vrna_pfl_fold_cb() <doxid-group__local__pf__fold_1ga8dc3788e66420f524f2bfc0b685a939d>` , :ref:`vrna_pfl_fold_up() <doxid-group__local__pf__fold_1ga1dd5c51b797c961124912e289bff553a>`

        double** vrna_pfl_fold_up (
	    const char* sequence,
	    int ulength,
	    int window_size,
	    int max_bp_span
	    )

    .. container:: brief-description

        Compute probability of contiguous unpaired segments.

    .. container:: detailed-description

        This is a simplified wrapper to :ref:`vrna_probs_window() <doxid-group__local__pf__fold_1ga70ec33810fa17a9f67a7c59f156d0137>` that given a nucleic acid sequence, a maximum length of unpaired segments ( ``ulength`` ), a window size, and a maximum base pair span computes the equilibrium probability of any segment not exceeding ``ulength`` . The probabilities to be unpaired are returned as a 1-based, 2-dimensional matrix with dimensions :math:` N \times M ` , where :math:`N` is the length of the sequence and :math:`M` is the maximum segment length. As an example, the probability of a segment of size 5 starting at position 100 is stored in the matrix entry :math:`X[100][5]` .
        
        It is the users responsibility to free the memory occupied by this matrix.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - sequence
        
                - The nucleic acid input sequence
        
            *
                - ulength
        
                - The maximal length of an unpaired segment
        
            *
                - window_size
        
                - The size of the sliding window
        
            *
                - max_bp_span
        
                - The maximum distance along the backbone between two nucleotides that form a base pairs
        
        
        
        .. rubric:: Returns:
        
        The probabilities to be unpaired for any segment not exceeding ``ulength``
        
        
        
        .. note::
        
            This function uses default model settings! For custom model settings, we refer to the function :ref:`vrna_probs_window() <doxid-group__local__pf__fold_1ga70ec33810fa17a9f67a7c59f156d0137>` .

    .. ref-code-block:: cpp
        :class: title-code-block

        :ref:`vrna_ep_t<doxid-group__struct__utils_1gab9ac98ab55ded9fb90043b024b915aca>`* pfl_fold (
	    char* sequence,
	    int winSize,
	    int pairSize,
	    float cutoffb,
	    double** pU,
	    :ref:`vrna_ep_t<doxid-group__struct__utils_1gab9ac98ab55ded9fb90043b024b915aca>`** dpp2,
	    FILE* pUfp,
	    FILE* spup
	    )

    .. container:: brief-description

        Compute partition functions for locally stable secondary structures.

    .. container:: detailed-description

        pfl_fold computes partition functions for every window of size 'winSize' possible in a RNA molecule, allowing only pairs with a span smaller than 'pairSize'. It returns the mean pair probabilities averaged over all windows containing the pair in 'pl'. 'winSize' should always be >= 'pairSize'. Note that in contrast to :ref:`Lfold() <doxid-group__local__mfe__fold_1ga16e5a70e60835bb969eaecbe6482f1be>` , bases outside of the window do not influence the structure at all. Only probabilities higher than 'cutoffb' are kept.
        
        If 'pU' is supplied (i.e is not the NULL pointer), :ref:`pfl_fold() <doxid-group__local__pf__fold_1ga7cdf690583871f9c158a0f91d4785cb6>` will also compute the mean probability that regions of length 'u' and smaller are unpaired. The parameter 'u' is supplied in 'pup[0][0]'. On return the 'pup' array will contain these probabilities, with the entry on 'pup[x][y]' containing the mean probability that x and the y-1 preceding bases are unpaired. The 'pU' array needs to be large enough to hold n+1 float* entries, where n is the sequence length.
        
        If an array dpp2 is supplied, the probability of base pair (i,j) given that there already exists a base pair (i+1,j-1) is also computed and saved in this array. If pUfp is given (i.e. not NULL), pU is not saved but put out imediately. If spup is given (i.e. is not NULL), the pair probabilities in pl are not saved but put out imediately.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - sequence
        
                - RNA sequence
        
            *
                - winSize
        
                - size of the window
        
            *
                - pairSize
        
                - maximum size of base pair
        
            *
                - cutoffb
        
                - cutoffb for base pairs
        
            *
                - pU
        
                - array holding all unpaired probabilities
        
            *
                - dpp2
        
                - array of dependent pair probabilities
        
            *
                - pUfp
        
                - file pointer for pU
        
            *
                - spup
        
                - file pointer for pair probabilities
        
        
        
        .. rubric:: Returns:
        
        list of pair probabilities
