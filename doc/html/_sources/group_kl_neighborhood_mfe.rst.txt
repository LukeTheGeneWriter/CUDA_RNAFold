        :ref:`vrna_sol_TwoD_t<doxid-structvrna__sol___two_d__t>`* vrna_mfe_TwoD (
	    :ref:`vrna_fold_compound_t<doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>`* vc,
	    int distance1,
	    int distance2
	    )

    .. container:: brief-description

        Compute MFE's and representative for distance partitioning.

    .. container:: detailed-description

        This function computes the minimum free energies and a representative secondary structure for each distance class according to the two references specified in the datastructure 'vars'. The maximum basepair distance to each of both references may be set by the arguments 'distance1' and 'distance2', respectively. If both distance arguments are set to '-1', no restriction is assumed and the calculation is performed for each distance class possible.
        
        The returned list contains an entry for each distance class. If a maximum basepair distance to either of the references was passed, an entry with k=l=-1 will be appended in the list, denoting the class where all structures exceeding the maximum will be thrown into. The end of the list is denoted by an attribute value of :ref:`INF <doxid-energy__const_8h_1a12c2040f25d8e3a7b9e1c2024c618cb6>` in the k-attribute of the list entry.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - vc
        
                - The datastructure containing all precomputed folding attributes
        
            *
                - distance1
        
                - maximum distance to reference1 (-1 means no restriction)
        
            *
                - distance2
        
                - maximum distance to reference2 (-1 means no restriction)
        
        
        
        .. rubric:: Returns:
        
        A list of minimum free energies (and corresponding structures) for each distance class
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_fold_compound_TwoD() <doxid-group__fold__compound_1ga9d24b196753b4f9f03774f11661510f5>` , :ref:`vrna_fold_compound_free() <doxid-group__fold__compound_1gadded6039d63f5d6509836e20321534ad>` , :ref:`vrna_pf_TwoD() <doxid-group__kl__neighborhood__pf_1ga0bc3427689bd09da09b8b3094a27f836>` :ref:`vrna_backtrack5_TwoD() <doxid-group__kl__neighborhood__mfe_1ga15a96fc96f4f4c2e01a11b3e17d1ef43>` , :ref:`vrna_sol_TwoD_t <doxid-structvrna__sol___two_d__t>` , :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>`

        char* vrna_backtrack5_TwoD (
	    :ref:`vrna_fold_compound_t<doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>`* vc,
	    int k,
	    int l,
	    unsigned int j
	    )

    .. container:: brief-description

        Backtrack a minimum free energy structure from a 5' section of specified length.

    .. container:: detailed-description

        This function allows one to backtrack a secondary structure beginning at the 5' end, a specified length and residing in a specific distance class. If the argument 'k' gets a value of -1, the structure that is backtracked is assumed to reside in the distance class where all structures exceeding the maximum basepair distance specified in :ref:`vrna_mfe_TwoD() <doxid-group__kl__neighborhood__mfe_1ga243c288b463147352829df04de6a2f77>` belong to.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - vc
        
                - The datastructure containing all precomputed folding attributes
        
            *
                - j
        
                - The length in nucleotides beginning from the 5' end
        
            *
                - k
        
                - distance to reference1 (may be -1)
        
            *
                - l
        
                - distance to reference2
        
        
        
        .. note::
        
            The argument 'vars' must contain precalculated energy values in the energy matrices, i.e. a call to :ref:`vrna_mfe_TwoD() <doxid-group__kl__neighborhood__mfe_1ga243c288b463147352829df04de6a2f77>` preceding this function is mandatory!
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_mfe_TwoD() <doxid-group__kl__neighborhood__mfe_1ga243c288b463147352829df04de6a2f77>`

                - seq

            *
                - structure1
        
                - The first reference structure in dot-bracket notation
        
            *
                - structure2
        
                - The second reference structure in dot-bracket notation
        
            *
                - circ
        
                - A switch to indicate the assumption to fold a circular instead of linear RNA (0=OFF, 1=ON)
