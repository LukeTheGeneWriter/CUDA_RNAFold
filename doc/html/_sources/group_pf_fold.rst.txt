    .. container:: brief-description

        Get the mean base pair distance in the thermodynamic ensemble from a probability matrix.

    .. container:: detailed-description

        :math:`<d> = \sum_{a,b} p_a p_b d(S_a,S_b)`
        
        this can be computed from the pair probs :math:`p_ij` as
        
        :math:`<d> = \sum_{ij} p_{ij}(1-p_{ij})`
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - length
        
                - The length of the sequence
        
            *
                - pr
        
                - The matrix containing the base pair probabilities
        
        
        
        .. rubric:: Returns:
        
        The mean pair distance of the structure ensemble

    .. container:: brief-description

        Compute the partition function :math:`Q` for a given RNA sequence, or sequence alignment.

    .. container:: detailed-description

        If *structure* is not a NULL pointer on input, it contains on return a string consisting of the letters " . , | { } ( ) " denoting bases that are essentially unpaired, weakly paired, strongly paired without preference, weakly upstream (downstream) paired, or strongly up- (down-)stream paired bases, respectively. If the parameter calculate_bppm is set to 0 base pairing probabilities will not be computed (saving CPU time), otherwise after calculations took place :ref:`pr <doxid-fold__vars_8h_1ac98ec419070aee6831b44e5c700f090f>` will contain the probability that bases *i* and *j* pair.
        
        SWIG Wrapper Notes This function is attached as method **pf()** to objects of type *fold_compound*
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - vc
        
                - The fold compound data structure
        
            *
                - structure
        
                - A pointer to the character array where position-wise pairing propensity will be stored. (Maybe NULL)
        
        
        
        .. rubric:: Returns:
        
        The Gibbs free energy of the ensemble ( :math:`G = -RT \cdot \log(Q) ` ) in kcal/mol
        
        
        
        .. note::
        
            This function is polymorphic. It accepts :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` of type :ref:`VRNA_FC_TYPE_SINGLE <doxid-group__fold__compound_1gga01a4ff86fa71deaaa5d1abbd95a1447da7e264dd3cf2dc9b6448caabcb7763cd6>` , and :ref:`VRNA_FC_TYPE_COMPARATIVE <doxid-group__fold__compound_1gga01a4ff86fa71deaaa5d1abbd95a1447dab821ce46ea3cf665be97df22a76f5023>` .

            :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` , :ref:`vrna_fold_compound() <doxid-group__fold__compound_1ga6601d994ba32b11511b36f68b08403be>` , :ref:`vrna_pf_fold() <doxid-group__pf__fold_1gac4a2a74a79e49818bc35412a2b392c7e>` , :ref:`vrna_pf_circfold() <doxid-group__pf__fold_1ga87e5a77b6e50dd54e9d032a9b92973be>` , :ref:`vrna_fold_compound_comparative() <doxid-group__fold__compound_1gad6bacc816af274922b13d947f708aa0c>` , :ref:`vrna_pf_alifold() <doxid-group__consensus__pf__fold_1ga374e31a0f326b2c5da5b84e143a63f38>` , :ref:`vrna_pf_circalifold() <doxid-group__consensus__pf__fold_1gab70fe6c9a78b79cc5669881720926e1d>` , :ref:`vrna_db_from_probs() <doxid-group__struct__utils_1ga0c28c410a5ab22d6ab9c77a84e8d5b44>` , :ref:`vrna_exp_params() <doxid-group__energy__parameters_1gab1f3016f96aa96bff020cdd904605afa>` , :ref:`vrna_aln_pinfo() <doxid-group__aln__utils_1gaf6421a1318586c59fea6a127ed9f65f3>`

    .. container:: brief-description

        Compute Partition function :math:`Q` (and base pair probabilities) for an RNA sequence using a comparative method.

    .. container:: detailed-description

        This simplified interface to :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` computes the partition function and, if required, base pair probabilities for an RNA sequence using default options. Memory required for dynamic programming (DP) matrices will be allocated and free'd on-the-fly. Hence, after return of this function, the recursively filled matrices are not available any more for any post-processing.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - sequence
        
                - RNA sequence
        
            *
                - structure
        
                - A pointer to the character array where position-wise pairing propensity will be stored. (Maybe NULL)
        
            *
                - pl
        
                - A pointer to a list of :ref:`vrna_ep_t <doxid-group__struct__utils_1gab9ac98ab55ded9fb90043b024b915aca>` to store pairing probabilities (Maybe NULL)
        
        
        
        .. rubric:: Returns:
        
        The Gibbs free energy of the ensemble ( :math:`G = -RT \cdot \log(Q) ` ) in kcal/mol
        
        
        
        .. note::
        
            In case you want to use the filled DP matrices for any subsequent post-processing step, or you require other conditions than specified by the default model details, use :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` , and the data structure :ref:`vrna_fold_compound_t <doxid-group__fold__compound_1ga1b0cef17fd40466cef5968eaeeff6166>` instead.

        float pf_fold_par (
	    const char* sequence,
	    char* structure,
	    :ref:`vrna_exp_param_t<doxid-group__energy__parameters_1ga01d8b92fe734df8d79a6169482c7d8d8>`* parameters,
	    int calculate_bppm,
	    int is_constrained,
	    int is_circular
	    )

    .. container:: brief-description

        Compute the partition function :math:`Q` for a given RNA sequence.

    .. container:: detailed-description

        If *structure* is not a NULL pointer on input, it contains on return a string consisting of the letters " . , | { } ( ) " denoting bases that are essentially unpaired, weakly paired, strongly paired without preference, weakly upstream (downstream) paired, or strongly up- (down-)stream paired bases, respectively. If :ref:`fold_constrained <doxid-fold__vars_8h_1a0afc287c2464866d94858c39175154af>` is not 0, the *structure* string is interpreted on input as a list of constraints for the folding. The character "x" marks bases that must be unpaired, matching brackets " ( ) " denote base pairs, all other characters are ignored. Any pairs conflicting with the constraint will be forbidden. This is usually sufficient to ensure the constraints are honored. If the parameter calculate_bppm is set to 0 base pairing probabilities will not be computed (saving CPU time), otherwise after calculations took place :ref:`pr <doxid-fold__vars_8h_1ac98ec419070aee6831b44e5c700f090f>` will contain the probability that bases *i* and *j* pair.
        
        Deprecated Use :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` instead
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - sequence
        
                - The RNA sequence input
        
            *
                - structure
        
                - A pointer to a char array where a base pair probability information can be stored in a pseudo-dot-bracket notation (may be NULL, too)
        
            *
                - parameters
        
                - Data structure containing the precalculated Boltzmann factors
        
            *
                - calculate_bppm
        
                - Switch to Base pair probability calculations on/off (0==off)
        
            *
                - is_constrained
        
                - Switch to indicate that a structure contraint is passed via the structure argument (0==off)
        
            *
                - is_circular
        
                - Switch to (de-)activate postprocessing steps in case RNA sequence is circular (0==off)
        
        
        
        .. rubric:: Returns:
        
        The Gibbs free energy of the ensemble ( :math:`G = -RT \cdot \log(Q) ` ) in kcal/mol
        
        
        
        .. note::
        
            The global array :ref:`pr <doxid-fold__vars_8h_1ac98ec419070aee6831b44e5c700f090f>` is deprecated and the user who wants the calculated base pair probabilities for further computations is advised to use the function :ref:`export_bppm() <doxid-group__pf__fold_1gac5ac7ee281aae1c5cc5898a841178073>`
        
        
        
        .. admonition:: Post-Condition
        
            After successful run the hidden folding matrices are filled with the appropriate Boltzmann factors. Depending on whether the global variable :ref:`do_backtrack <doxid-group__model__details_1gad512b5dd4dbec60faccfe137bb474489>` was set the base pair probabilities are already computed and may be accessed for further usage via the :ref:`export_bppm() <doxid-group__pf__fold_1gac5ac7ee281aae1c5cc5898a841178073>` function. A call of :ref:`free_pf_arrays() <doxid-group__pf__fold_1gae73db3f49a94f0f72e067ecd12681dbd>` will free all memory allocated by this function. Successive calls will first free previously allocated memory before starting the computation.
        
        
        
        .. admonition:: See also:
        
            :ref:`vrna_pf() <doxid-group__pf__fold_1ga29e256d688ad221b78d37f427e0e99bc>` , :ref:`bppm_to_structure() <doxid-group__struct__utils_1ga129d81c4a1ead793c5b2311333e03dfa>` , :ref:`export_bppm() <doxid-group__pf__fold_1gac5ac7ee281aae1c5cc5898a841178073>` , :ref:`vrna_exp_params() <doxid-group__energy__parameters_1gab1f3016f96aa96bff020cdd904605afa>` , :ref:`free_pf_arrays() <doxid-group__pf__fold_1gae73db3f49a94f0f72e067ecd12681dbd>`

The character "x" marks bases that must be unpaired, matching brackets " ( ) " denote base pairs, all other characters are ignored. Any pairs conflicting with the constraint will be forbidden. This is usually sufficient to ensure the constraints are honored. If :ref:`do_backtrack <doxid-group__model__details_1gad512b5dd4dbec60faccfe137bb474489>` has been set to 0 base pairing probabilities will not be computed (saving CPU time), otherwise :ref:`pr <doxid-fold__vars_8h_1ac98ec419070aee6831b44e5c700f090f>` will contain the probability that bases *i* and *j* pair.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - sequence
        
                - The RNA sequence input
        
            *
                - structure
        
                - A pointer to a char array where a base pair probability information can be stored in a pseudo-dot-bracket notation (may be NULL, too)
        
        
        
        .. rubric:: Returns:
        
        The Gibbs free energy of the ensemble ( :math:`G = -RT \cdot \log(Q) ` ) in kcal/mol
        
        
        
        .. note::
        
            The global array :ref:`pr <doxid-fold__vars_8h_1ac98ec419070aee6831b44e5c700f090f>` is deprecated and the user who wants the calculated base pair probabilities for further computations is advised to use the function :ref:`export_bppm() <doxid-group__pf__fold_1gac5ac7ee281aae1c5cc5898a841178073>` .
        
            **OpenMP:** This function is not entirely threadsafe. While the recursions are working on their own copies of data the model details for the recursions are determined from the global settings just before entering the recursions. Consider using :ref:`pf_fold_par() <doxid-group__pf__fold_1gac4f95bee734b2563a3d6e9932117ebdf>` for a really threadsafe implementation.
        
        
        
        .. admonition:: Pre-Condition
        
            This function takes its model details from the global variables provided in *RNAlib*
        
        
        
        .. admonition:: Post-Condition
        
            After successful run the hidden folding matrices are filled with the appropriate Boltzmann factors. Depending on whether the global variable :ref:`do_backtrack <doxid-group__model__details_1gad512b5dd4dbec60faccfe137bb474489>` was set the base pair probabilities are already computed and may be accessed for further usage via the :ref:`export_bppm() <doxid-group__pf__fold_1gac5ac7ee281aae1c5cc5898a841178073>` function. A call of :ref:`free_pf_arrays() <doxid-group__pf__fold_1gae73db3f49a94f0f72e067ecd12681dbd>` will free all memory allocated by this function. Successive calls will first free previously allocated memory before starting the computation.
        
        
        
        .. admonition:: See also:
        
            :ref:`pf_fold_par() <doxid-group__pf__fold_1gac4f95bee734b2563a3d6e9932117ebdf>` , :ref:`pf_circ_fold() <doxid-group__pf__fold_1ga819ce5fca8984004ac81c4a3b04cb735>` , :ref:`bppm_to_structure() <doxid-group__struct__utils_1ga129d81c4a1ead793c5b2311333e03dfa>` , :ref:`export_bppm() <doxid-group__pf__fold_1gac5ac7ee281aae1c5cc5898a841178073>`
