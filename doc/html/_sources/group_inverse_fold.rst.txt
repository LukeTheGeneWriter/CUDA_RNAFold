        float inverse_fold (
	    char* start,
	    const char* target
	    )

    .. container:: brief-description

        Find sequences with predefined structure.

    .. container:: detailed-description

        This function searches for a sequence with minimum free energy structure provided in the parameter 'target', starting with sequence 'start'. It returns 0 if the search was successful, otherwise a structure distance in terms of the energy difference between the search result and the actual target 'target' is returned. The found sequence is returned in 'start'. If :ref:`give_up <doxid-group__inverse__fold_1ga7ec4ba51f86e1717a1e174264e4a75ce>` is set to 1, the function will return as soon as it is clear that the search will be unsuccessful, this speeds up the algorithm if you are only interested in exact solutions.
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - start
        
                - The start sequence
        
            *
                - target
        
                - The target secondary structure in dot-bracket notation
        
        
        
        .. rubric:: Returns:
        
        The distance to the target in case a search was unsuccessful, 0 otherwise

        float inverse_pf_fold (
	    char* start,
	    const char* target
	    )

    .. container:: brief-description

        Find sequence that maximizes probability of a predefined structure.

    .. container:: detailed-description

        This function searches for a sequence with maximum probability to fold into the provided structure 'target' using the partition function algorithm. It returns :math:`-kT \cdot \log(p)` where :math:`p` is the frequency of 'target' in the ensemble of possible structures. This is usually much slower than :ref:`inverse_fold() <doxid-group__inverse__fold_1ga7af026de55d4babad879f2c92559cbbc>` .
        
        
        
        .. rubric:: Parameters:
        
        .. list-table::
            :widths: 20 80
        
            *
                - start
        
                - The start sequence
        
            *
                - target
        
                - The target secondary structure in dot-bracket notation
        
        
        
        .. rubric:: Returns:
        
        The distance to the target in case a search was unsuccessful, 0 otherwise
